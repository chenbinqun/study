#<center>CSS 单位</center>

## CSS 长度值

* em: 相对于父元素的字体大小
* ex: 相对于小写字符 `x` 的高度
* gd：一般用在东亚字体排版上，这个与英文并无关系
* rem：相对于根元素字体大小
* vw：相对于视窗的宽度；视窗宽度为100vw
* vh：相对于视窗的高度；视窗高度为100vh
* vm：相对于视察的宽度和高度，取决于哪个更小
* ch：相对于0尺寸
* px：相当于视窗分辨率而不是视窗大小；通常为一个点或1/72英寸
* in：英寸
* cm：厘米
* mm：毫米
* pt：1/7英寸
* pc：12点活字，或1/12点
* %：相对于父元素

## 时间、频率、角度

* deg：角度
* grad：百分度
* rad： 弧度
* turn：圈数
* ms：毫秒数
* s：秒数
* Hz：赫兹
* kHz：千赫
